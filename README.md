O projeto ScannerExpert é uma aplicação de console em C# que simula uma varredura de sistema com o objetivo de aplicar a 1ª Heurística de Nielsen, chamada Visibilidade do Status do Sistema. Durante a execução, o programa informa continuamente o usuário sobre o que está acontecendo, exibindo mensagens de progresso em tempo real, como verificação de CPU, leitura de memória e outras etapas. Isso evita que o usuário pense que o sistema travou, garantindo uma comunicação clara e eficiente ao longo de todo o processo.
